# AI-Based Attendance Tracker and Predictor

## 1. Overview

This is a Python application that tracks attendance and predicts future attendance using basic logic and a simple machine learning model.

---

## 2. Problem

Students may fall below required attendance due to lack of tracking.

---

## 3. Features

* Add and manage subjects
* View attendance percentage
* Basic prediction (bunk or required classes)
* AI-based prediction
* Risk classification

---

## 4. Tech Stack

* Python
* NumPy
* Scikit-learn
* JSON
